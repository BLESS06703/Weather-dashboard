## 📱 Features

### 🌤️ Real-Time Weather
- Live weather data for any city worldwide
- Current temperature, feels like, humidity, wind speed, pressure
- Beautiful weather icons with condition descriptions
- Auto-refresh with customizable intervals

### 📊 5-Day Forecast
- Detailed weather predictions for the next 5 days
- Daily high/low temperatures
- Humidity and weather condition forecasts
- Visual forecast cards with icons

### 🗺️ Interactive Global Map
- Visualize weather data on an interactive world map
- City markers with live weather information
- Multiple map layers (temperature, precipitation, wind)
- Zoom and pan functionality

### 📈 Analytics & Charts
- Temperature trend charts
- Humidity distribution graphs
- Historical data visualization
- Interactive Chart.js graphs

### 🎨 Customization
- Dark/Light theme with auto-detect
- Temperature units (Celsius/Fahrenheit)
- Custom logo upload (text, image, or emoji)
- Adjustable auto-refresh intervals
- Data saver mode for mobile networks

### 💾 Data Management
- Export weather data as JSON
- Persistent storage with localStorage
- Clear all data option
- Import/export functionality
